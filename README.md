# OpenSource-stuff-
Compact Roblox UI library with dynamic z-index windows, touch support, and a remade notification system.

## Latest upgrades
- Smaller default window layout (`480x340`) for mobile and compact screens.
- Slimmer sidebar/topbar design for a cleaner modern look.
- Advanced notifications with:
  - Label aliases (`Title`, `Label`, `Labels`)
  - Message aliases (`Message`, `Text`)
  - Corner positioning (`TopRight`, `TopLeft`, `BottomRight`, `BottomLeft`)
  - Optional action button callbacks
  - Optional click-to-dismiss behavior
